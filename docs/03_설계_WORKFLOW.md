## 1. 문서 연결 지도 — 조건(1~4장)이 절차(5~8장)로 흐르는 방식

```mermaid
flowchart LR
    subgraph COND["① 설계조건 · 표준화 (1~4장)"]
        A1["1장 재료<br/>Fy, Fu / F10T·F13T"]
        A2["2장 일반사항<br/>볼트장력 To / 미끄럼·지압강도 / 연단거리"]
        A3["3장 표준화<br/>첨판폭=공칭폭 / 피치·배치 고정"]
        A4["4장 원칙<br/>전강도·부분강도 / 공칭 휨·압축강도"]
    end
    subgraph PROC["② 설계 프로시저 (5~8장)"]
        P["플랜지 이음 + 웨브 이음<br/>첨판·볼트 결정"]
    end
    A1 --> P
    A2 --> P
    A3 --> P
    A4 --> P
    COND -->|"입력값·규칙 제공"| PROC
```

**요점**: 1~4장은 "재료·강도·표준치수·소요강도 기준"이라는 **입력값과 규칙**을 확정하고, 5~8장은 그 값을 넣어 **첨판 두께·볼트 개수·배열**을 계산한다.

## 2. 시작점 — 접합 유형 선택 결정 트리

```mermaid
flowchart TD
    S(["설계 시작"]) --> Q1{"부재 종류?"}
    Q1 -->|"보 (휨 지배)"| B{"접합 방식?"}
    Q1 -->|"기둥 (압축 지배)"| C{"접합 방식?"}
    B -->|"마찰"| B5["▶ 5장 보 마찰접합<br/>α = 1.0 / 0.85 / 0.7"]
    B -->|"지압"| B6["▶ 6장 보 지압접합<br/>α = 1.0 / 0.85 / 0.7"]
    C -->|"마찰"| C7["▶ 7장 기둥 마찰접합<br/>α = 1.0 / 0.75 / 0.5"]
    C -->|"지압"| C8["▶ 8장 기둥 지압접합<br/>α = 1.0 / 0.75 / 0.5"]
```

| 판단 | 결과 | 이유 |
|---|---|---|
| 부재 = 보 | 휨모멘트로 소요력 산정 | 4.1 |
| 부재 = 기둥 | 압축력으로 소요력 산정 | 4.2 |
| 마찰 선택 | 볼트강도 = 설계미끄럼강도(φ=0.85) | 2.4 |
| 지압 선택 | 볼트강도 = 설계지압강도(φ=0.75, 3모드 최솟값) | 2.5·2.6 |
| 지압 선택 시 | 하중비 < 1.43이면 **사용성 마찰접합 별도 검토** | 2.8 |

## 3. 【5장】 보 마찰접합 — 표준 절차 (다른 장의 기준)

```mermaid
flowchart TD
    ST(["보 마찰접합 시작"]) --> SL["세장비 검토<br/>플랜지 λf, 웨브 λw"]
    SL --> FL[["플랜지 이음"]]
    SL --> WB[["웨브 이음"]]

    subgraph FLG["플랜지 이음"]
        F1["① 공칭휨강도 Mn<br/>국부좌굴 최솟값"] --> F2["② 소요휨강도 Mu = α·φ·Mn<br/>→ 소요축력 Puf = Mu / dm"]
        F2 --> F3["③ 첨판 소요단면적<br/>Aupf = Puf / (φ·Fy)"]
        F3 --> F4["④ 첨판 폭(=공칭폭)·볼트지름·열수 m<br/>표 5.1"]
        F4 --> F5["⑤ 첨판 두께<br/>외 0.5Aupf/폭, 내 0.5Aupf/2폭"]
        F5 --> F6["⑥ 설계미끄럼강도 φRn<br/>= φ·μ·hsc·To·Ns"]
        F6 --> F7["⑦ 요구행수 n = Puf/(m·φRn) 올림<br/>개수 = m×n"]
        F7 --> F8["⑧ 첨판 두께 조정"]
        F8 --> F9["⑨ 첨판 길이 lpf"]
    end

    subgraph WEB["웨브 이음"]
        W1["① 소요전단강도<br/>Vu = α·φv·0.6·Fy·Aw"] --> W2["② 볼트지름(=플랜지와 동일)"]
        W2 --> W3["③ 설계미끄럼강도 φRn<br/>→ 요구개수 mn = Vuw/φRn"]
        W3 --> W4{"④ 배치 검토<br/>(m-1)Pc + (60+tf+r)·2 ≤ H ?"}
        W4 -->|"불만족"| W4b["행수↑ 또는 피치 Pc 조정<br/>Pc = 60/90/120"]
        W4b --> W4
        W4 -->|"만족"| W5["⑤ 첨판 춤 dpw·두께 tpw"]
        W5 --> W6["⑥ 첨판 너비 w.pw"]
    end
    F9 --> DONE(["완료"])
    W6 --> DONE
```

**단계 설명 (플랜지)**

| 단계 | 입력 | 계산/판정 | 출력 |
|---|---|---|---|
| ① Mn | Zx, Sx, Fy, λ | 5.2.1·5.2.2 중 작은 Mn | 공칭휨강도 |
| ② Mu·Puf | Mn, α, dm | Mu=α·φ·Mn, Puf=Mu/dm | 플랜지 소요축력 |
| ③ Aupf | Puf, Fy | Puf/(φ·Fy), φ=0.9 | 소요단면적 |
| ④ 폭·m | 공칭폭 | 표 5.1 | 볼트 규격·열수 |
| ⑤ 두께 | Aupf, 폭 | 외·내 두께식 | 첨판 두께 |
| ⑥ φRn | To, μ | 표 1.7 ×0.85 | 볼트 단위강도 |
| ⑦ n | Puf, m, φRn | 올림(300mm=0.5단위) | 행수·개수 |
| ⑧ 조정 | 볼트강도 | 외/내 재배분 | 확정 두께 |
| ⑨ 길이 | n | 정렬/엇모식 | 첨판 길이 |

**단계 설명 (웨브)**: ① 전단으로 Vu 산정 → ③ 요구개수 → **④ 배치조건 루프**(행수 적게·피치 크게 우선, 춤≤200mm면 엇갈림) → ⑤⑥ 첨판.

## 4. 【6장】 보 지압접합 — 마찰과의 차이 강조 (웨브 재산정 루프)

```mermaid
flowchart TD
    ST(["보 지압접합 시작"]) --> FL[["플랜지 이음"]]
    ST --> WB[["웨브 이음 (가정→검토 루프)"]]

    subgraph FLG["플랜지 이음 (5장과 동일, 볼트강도만 지압)"]
        F1["①~⑤ = 5장과 동일<br/>Mu → Puf → 첨판 폭·두께"] --> F6["⑥ 설계지압강도<br/>= min(볼트전단, 모재지압, 첨판지압)×0.75"]
        F6 --> F7["⑦ 요구행수 n<br/>(3모드 최댓값 올림)"]
        F7 --> F8["⑧ 두께 조정 → ⑨ 길이"]
    end

    subgraph WEB["웨브 이음"]
        W1["① 소요전단강도 Vuw"] --> W2["② 볼트지름 선정 · 배열 가정"]
        W2 --> W3["③ 웨브첨판 춤·두께 가정"]
        W3 --> W4["④ 설계지압강도 → 요구 열수<br/>(모재 웨브는 연단항 Rn1 제외)"]
        W4 --> W5{"⑤ 가정 배열과 일치?"}
        W5 -->|"불일치"| W2
        W5 -->|"일치"| W6["⑥ 첨판 춤·두께 결정 → ⑦ 너비"]
    end
    F8 --> DONE(["완료"])
    W6 --> DONE
```

| 마찰(5장)과 다른 점 | 내용 |
|---|---|
| 볼트 단위강도 | 미끄럼강도 → **지압강도**(3모드 최솟값 × 0.75) |
| 플랜지 요구행수 | 볼트전단·모재지압·첨판지압 **3종 중 최대** |
| 웨브 설계 | 직접 산정 → **"가정→검토→재산정" 루프** |
| 모재 웨브 | 연단부 없음 → 연단인접 지압항(Rn1) 제외 |

## 5. 【7장】 기둥 마찰접합 — 휨 대신 압축

```mermaid
flowchart TD
    ST(["기둥 마찰접합 시작"]) --> SL["세장비 검토"]
    SL --> FL[["플랜지 이음"]]
    SL --> WB[["웨브 이음"]]

    subgraph FLG["플랜지 이음"]
        F1["① 플랜지 소요압축강도<br/>Puf = α·φc·bf·tf·Fy"] --> F2["② 첨판 소요단면적 → ③ 폭·볼트·열수"]
        F2 --> F3["④ 첨판 두께"]
        F3 --> F4["⑤ 설계미끄럼강도 φRn → ⑥ 요구행수 n"]
        F4 --> F5["⑦ 두께 조정 → ⑧ 길이"]
    end

    subgraph WEB["웨브 이음"]
        W1["① 웨브 소요압축강도<br/>압연: α·φc·(H·tw+0.86r²)·Fy<br/>용접: α·φc·H·tw·Fy"] --> W2["② 볼트지름 → ③ 요구개수 mn"]
        W2 --> W3{"④ 배치 검토<br/>(m-1)Pc+(60+tf+r)·2 ≤ H ?"}
        W3 -->|"불만족"| W3b["행수/피치 조정"]
        W3b --> W3
        W3 -->|"만족"| W4["⑤ 첨판 춤·두께 → ⑥ 너비"]
    end
    F5 --> DONE(["완료"])
    W4 --> DONE
```

| 보(5장)와 다른 점 | 내용 |
|---|---|
| 소요력 | 휨 Mu 대신 **압축 Puf = α·φc·bf·tf·Fy** |
| 웨브 소요력 | **압축**(압연=필렛 포함 / 용접=미포함) |
| 부분강도비 α | **1.0 / 0.75 / 0.5** (밀착접합 고려) |
| 이격거리 | 밀착접합(metal touch) 시 **0mm** |
| 나머지(첨판·볼트·배열) | 5장과 **동일** |

## 6. 【8장】 기둥 지압접합 — 압축(7장) + 지압(6장)

```mermaid
flowchart TD
    ST(["기둥 지압접합 시작"]) --> FL[["플랜지 이음"]]
    ST --> WB[["웨브 이음 (가정→검토 루프)"]]

    subgraph FLG["플랜지 이음"]
        F1["① 플랜지 소요압축강도<br/>Puf = α·φc·bf·tf·Fy"] --> F2["②~④ 첨판 소요단면적·폭·두께"]
        F2 --> F3["⑤ 설계지압강도 min(3모드)×0.75<br/>→ ⑥ 요구행수(3모드 최대)"]

    end

    subgraph WEB["웨브 이음"]
        W1["① 웨브 소요압축강도<br/>압연=필렛포함 / 용접=미포함"] --> W2["② 볼트지름·배열 가정 → ③ 첨판 춤·두께 가정"]
        W2 --> W3["④ 설계지압강도 → 요구 열수<br/>(모재 웨브 연단항 제외)"]

        W4 -->|"불일치"| W2
        W4 -->|"일치"| W5["⑥ 춤·두께 결정 → ⑦ 너비"]
    end
    F4 --> DONE(["완료"])
    W5 --> DONE
```

**8장 = 7장의 압축 소요력 + 6장의 지압 볼트강도·웨브 루프** 조합. 부분강도비 α = 1.0 / 0.75 / 0.5.

## 7. 빠른 참조 — 설계 완료 전 체크리스트

**입력 조건 (1~4장)**
- [ ] 강종 확정, `Fy`·`Fu` 확인 (표 1.2) — 첨판은 모재와 동일 강종
- [ ] 볼트 등급(F10T/F13T)·지름(M16/M20/M22) 선정, 플랜지=웨브 동일 지름
- [ ] 설계볼트장력 `To`(표 1.6), 마찰이면 미끄럼강도(표 1.7)/지압이면 강도(표 1.8) 확보
- [ ] 부재 구분(보=휨 / 기둥=압축), 부분강도비 α 결정
- [ ] 접합방식(마찰/지압) 결정, 지압이면 **하중비 ≥ 1.43** 또는 사용성 마찰 별도검토(2.8)

**표준 치수 (3장)**
- [ ] 플랜지 첨판 폭 = 형강 공칭폭, 첨판 두께 = 표 3.1/3.2 규격값
- [ ] 피치: 정렬 60 / 엇모 45mm, 웨브 상하피치 60·90·120mm
- [ ] 연단거리 40mm, 볼트중심 응력방향거리 60mm, 이격 10mm(밀착 0)

**플랜지 이음**
- [ ] 소요축력 `Puf` 산정 (보: Mu/dm / 기둥: α·φc·bf·tf·Fy)
- [ ] 볼트 열수 m(표 5.1/7.1), 요구행수 n (300mm는 0.5단위 올림)
- [ ] 외·내첨판 두께 조정, 첨판 길이 산정

**웨브 이음**
- [ ] 소요전단(보) / 소요압축(기둥) 산정
- [ ] 배치조건 `(m-1)Pc+(60+tf+r)·2 ≤ H` 만족, 행수 적게·피치 크게 우선
- [ ] 춤 ≤ 200mm면 웨브/플랜지 볼트 절반피치 엇갈림
- [ ] (지압) 가정 배열 = 설계 배열 일치 확인(불일치 시 재수행)
- [ ] 웨브 첨판 춤 ≥ 부재 춤의 60%, 두께·너비 확정

**최종 검토**
- [ ] 모든 부재(첨판)·볼트가 표준 규격 범위 내인지
- [ ] 과대·슬롯구멍, 나사부 전단면 포함 등 **표준접합 적용 제외 조건** 없는지
