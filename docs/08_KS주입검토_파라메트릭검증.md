# KS 단면성능 주입 검토 · 파라메트릭 골든 확장 검증

## 1. KS D 3502 정밀 단면성능 주입 — 완료 (원표 반영)

현대제철 Products Guide(KS D 3502:2007 / JIS G 3192:1990) 원표에서 **73종 중 70종**의 확정 단면성능을 추출·주입했다(`src/engine/ks_d3502.ts`, propSource='ks'). **단위 변환**: A(cm²)×100 = mm², Sx·Zx(cm³)×1000 = mm³, r = mm. Sx=탄성·Zx=소성 단면계수.
- 검증: H-386x299 Zx=1,920,000 = 제9장 예제값 정확 일치. r도 KS값(캘리브레이션 제거).
- KS 미수록 보강단면 3종(304x301, 310x305, 388x402)만 편람 Mu 캘리브레이션 유지.

**단, 검증률(9.0 경계)은 변하지 않았다** (보 69/73·기둥 50/73 동일). 이유:

| 근거 | 설명 |
|---|---|
| 기둥 소요력이 기하 기반 | `Nc = α·φc·bf·tf·Fy`, `Vw = α·φc·H·tw·Fy` — **bf·tf·H·tw(치수)** 만 사용, Zx/Sx(단면성능) 무관. 따라서 KS Zx 주입해도 기둥 첨판 두께 불변 |
| 순단면 파단검토 추가 실험 | 첨판 두께에 순단면 파단(φ=0.75) 검토를 추가하니 **대다수 과대**(보 전항목 69→26) → 기각 |
| 보 플랜지는 이미 72/73 | Mu 기반 보 플랜지는 이미 최고 수준. Zx 미세보정 여지 없음(유일 예외 H-388x402 광폭박판 이상치) |

**9.0 경계의 실제 성격**: 계산 두께가 정확히 9.0mm 등 등급 경계에 놓일 때 부록은 상위 등급(12)으로 올림. 이는 부록이 사용한 **정확한 KS 단면적/반올림 세칙**과의 미세차이(±1등급)이며, 현재 확보 불가능한 KS 원표 값이 있어야 재현 가능. **실무상 ±1등급 안전측**이므로 그대로 둔다.

## 2. 파라메트릭 골든 확장 검증 — 성공

엔진은 조건(α·강종·볼트)을 파라미터로 받으므로, 대표 조건을 추가 검증해 **일반화**를 확인:

| 조건 | 시험 대상 | Mu | 외첨판 | 웨브 | 전항목 |
|---|---|---|---|---|---|
| **보 85% SHN490 F10T** (마찰, 1/4) | 강도비 α=0.85 | 19/19 | **19/19** | 17/19 | **17/19** |
| **보 100% SHN490 F13T** (마찰, 1/4) | 볼트등급 F13T | 19/19 | **19/19** | 18/19 | **18/19** |

### 확인 사항
- **α 스케일**: 85% 휨모멘트 = 100%의 0.85배 정확 재현(H-200x200: 154→131). 볼트·첨판 이산결과도 일치.
- **F13T**: 휨모멘트는 동일(단면 능력)하나 볼트가 강해 **개수 감소**(H-125x125: F10T 2×5 → F13T 2×4) — 엔진이 정확히 반영.
- **플랜지 19/19**(양 조건) → 소요력·첨판·볼트 설계흐름이 조건 무관하게 일반화됨.
- 잔여 1~2건은 초소형 웨브(H-100x100 1×2 vs 1×1) 경계로, 100% 검증의 잔여와 동일 성격.

### 결론
- 100% SHN490 F10T(69/73) 코어 로직이 **α·볼트등급 변경에도 동일 품질로 재현**됨을 확인.
- 전 조건(SS400, 70/75/50%, F13T×강종) 골든 전수 전사는 방대하나(≈20 표세트), 엔진이 파라메트릭이므로 대표 검증으로 **일반화가 입증**됨.
- 검증 스크립트: `src/engine/validate_variants.ts` (`node src/engine/validate_variants.ts`).
