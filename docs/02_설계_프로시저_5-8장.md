> **원전**: 한국강구조학회 『고력볼트 표준접합 설계편람』 (KSSC-M-09-03)

## 0. 전체 구조 — 4가지 접합 유형

| 장 | 부재 | 접합방식 | 지배력 | 볼트강도 기준 | 부분강도비 α |
|---|---|---|---|---|---|
| **5장** | 보(Beam) | 마찰(Friction) | 휨(M) | 설계미끄럼강도 φRn (φ=0.85) | 1.0 / 0.85 / 0.7 |
| **6장** | 보(Beam) | 지압(Bearing) | 휨(M) | 설계지압강도 (φ=0.75) | 1.0 / 0.85 / 0.7 |
| **7장** | 기둥(Column) | 마찰 | 압축(P) | 설계미끄럼강도 φRn (φ=0.85) | 1.0 / 0.75 / 0.5 |
| **8장** | 기둥 | 지압 | 압축(P) | 설계지압강도 (φ=0.75) | 1.0 / 0.75 / 0.5 |

**공통 원칙**
- 모든 유형은 **플랜지 이음**과 **웨브 이음**을 각각 설계한다.
- 플랜지 볼트와 웨브 볼트의 **지름은 동일**하다 → 웨브 볼트의 단위강도(미끄럼/지압)는 플랜지와 같다.
- 마찰↔지압, 보↔기둥의 차이는 **① 소요력을 구하는 방법(휨 vs 압축)** 과 **② 볼트 단위강도 산정식(미끄럼 vs 지압)** 두 곳에 집중된다. 나머지(첨판 폭/두께/길이, 배열)는 거의 공유된다.

## 제5장 보의 마찰접합부 설계 (기준 절차)

### 5.1 설계 순서 개요
사전: **플랜지·웨브의 세장비(판폭두께비) 검토** → 공칭강도 결정 방식 선택.

**[플랜지 이음]** ① 소요휨강도 → ② 플랜지 소요축력 → ③ 첨판 소요단면적 → ④ 첨판 폭·볼트지름·열수 → ⑤ 첨판 두께 → ⑥ 설계미끄럼강도·요구행수 → ⑦ 볼트 배열·개수 → ⑧ 첨판 두께 조정 → ⑨ 첨판 길이

**[웨브 이음]** ① 소요전단강도 → ② 볼트지름 → ③ 설계미끄럼강도·요구개수 → ④ 웨브 춤에 따른 볼트 배열 → ⑤ 첨판 춤·두께 → ⑥ 첨판 너비

---

### [플랜지 이음]

**5.2 접합부 소요휨강도와 플랜지 소요축력**

**5.2.1 플랜지 국부좌굴 관련 공칭휨강도** — `λf = bf/(2·tf)`
- `λf ≤ λpf` : `Mn = Mp = Zx·Fy` (콤팩트)
- `λpf < λf ≤ λrf` : `Mn = Mp − (Mp − 0.7Fy·Sx)·(λf−λpf)/(λrf−λpf)` (비콤팩트)
- `λf > λrf` : `Mn = 0.9·E·kc·Sx / λf²` (세장)
- `λpf = 0.38·√(E/Fy)`

**5.2.2 웨브 국부좌굴 관련 공칭휨강도** — `λw = (H − 2r − 2tf)/tw`
- `λw ≤ λpw` : `Mn = Mp = Zx·Fy`
- `λpw < λw ≤ λrw` : 선형보간식
- `λpw = 3.76·√(E/Fy)`, `λrw = 5.70·√(E/Fy)`

**5.2.3 접합부 소요휨강도** — 5.2.1·5.2.2 중 **작은 Mn** 선택
> `Mu = α · φ · Mn`  (α = 1.0/0.85/0.7, φ = 0.9)

**5.2.4 플랜지 소요축력** — 휨을 짝힘(couple)으로 치환
> `Puf = Mu / dm`
> `dm` = (내·외첨판 모두 사용 시) 상·하 플랜지 중심 간 거리 / (외첨판만) H형강 전체 춤

**5.3 플랜지 첨판 폭 및 두께**
- **5.3.1 소요단면적**: `Aupf = Puf / (φ·Fy)`  (φ=0.9, Fy=첨판 항복강도)
- **5.3.2 첨판 폭**: = 플랜지 **공칭폭** (표 3.3에서 내·외첨판 폭 결정)
- **5.3.3 첨판 두께**(외·내첨판 동일 응력 분담 시):
  - 외첨판 두께 = `0.5·Aupf / 외첨판 폭`
  - 내첨판 두께 = `0.5·Aupf / (2 × 내첨판 폭)`

**5.4 플랜지 고력볼트 설계미끄럼강도와 배열**
- **5.4.1 설계미끄럼강도**: `φRn = φ·μ·hsc·To·Ns` (φ=0.85, μ=0.5, hsc=1.0, To=설계볼트장력, Ns=전단면수) → [표 1.7]
- **5.4.2 볼트 열수 m**: 플랜지 공칭폭에 따라 [표 5.1]로 결정

  **[표 5.1] 플랜지 공칭폭별 볼트 배치**

  | 공칭폭 | 볼트 규격 | 열수 | 배치 |
  |---|---|---|---|
  | 100, 125 | M16 | 2 | 정렬 |
  | 150, 175, 200, 250 | M20 | 2 | 정렬 |
  | 300 | M22 | 4 | 엇모 |
  | 350, 400 | M22 | 4 | 정렬 |

- **5.4.3 요구 행수 n**: `n = Puf / (m × φRn)` 을 **올림**한 자연수
  - 단, **공칭폭 300mm(엇모)** 는 자연수가 아닌 **0.5단위로 올림**
  - 볼트 개수 = `m × n` (열수 × 행수)
  - 외첨판 부담 행수 `no = n`, 내첨판 부담 행수 `ni = 2·no − n`

**5.5 플랜지 첨판 두께 조정 및 길이**
- **5.5.1 두께 조정**: 설계된 볼트의 1면전단 설계강도만큼 외첨판이 부담, 나머지를 내첨판이 부담하도록 두께 재산정. (외·내 두께차가 작은 조합 선택)
- **5.5.2 첨판 길이**
  - 정렬(공칭폭 ≠ 300): `lpf = 2·[(n−1)·60 + 2·40] + 10`
  - 엇모(공칭폭 = 300): `lpf = 2·[(2n−1)·45 + 2·40] + 10`
  - (형강 이음 틈 10mm 포함)

### [웨브 이음]

**5.6 접합부 소요전단강도** — 판폭두께비 무시, 공칭전단강도 기준
> `Vu = α · φv · 0.6·Fy·Aw`  (α = 1.0/0.85/0.7, φv = 0.9, Aw = H × tw)

**5.7 웨브 고력볼트 설계미끄럼강도** — 볼트지름이 플랜지와 같으므로 **φRn = 플랜지와 동일**

**5.8 웨브 이음 볼트 개수와 배열**
- **5.8.1 요구개수**: `mn = Vuw / (φRn)`
- **5.8.2 웨브 춤에 따른 배열 검토** — 1행/2행/3행 순차 검토, 피치 `Pc = 60/90/120`
  > 배치 성립 조건: `(m−1)·Pc + (60 + tf + r)·2 ≤ H`  (m=열수, r=필렛반경, tf=플랜지 두께, H=전체 춤)
  - **선택 규칙**: ① 볼트 **행수가 적은 것** 우선 ② 동일 행수면 **최대 피치**(웨브 춤을 꽉 채움) ③ 웨브 춤 ≤ 200mm면 웨브/플랜지 볼트를 **절반 피치 엇갈림**([그림 3.4])

**5.9 웨브 첨판 춤·두께·너비**
- **5.9.1 춤**: `dpw = (m−1)·Pc + 2·40`
- **5.9.2 두께**(양면 동일): `tpw = 0.5·Vuw / (φv·0.6·Fy·dpw)`
- **5.9.3 너비**: `w.pw = 2·[(n−1)·60 + 2·40] + 10`

## 제6장 보의 지압접합부 설계

**5장과 대부분 동일**하며, **볼트 강도만 미끄럼강도 → 지압강도**로 바뀐다. 웨브 이음은 "가정→검토→재산정" 루프 방식이라는 점이 다르다.

### 6.1 설계 순서 개요
**[플랜지 이음]** 5장과 동일하되 ⑥에서 **설계지압강도·요구행수** 사용
**[웨브 이음]** ① 소요전단강도 → ② 볼트지름 선정·**배열 가정** → ③ 웨브첨판 춤·두께 **가정** → ④ 설계지압강도·요구 열수 → ⑤ **배열 검토(가정과 다르면 ②부터 재수행)** → ⑥ 첨판 춤·두께 결정 → ⑦ 첨판 너비

### 6.2 / 6.3 / 6.5 / 6.6 — **마찰접합(5장)과 동일**
- 6.2 소요휨강도·소요축력 = 5.2와 동일
- 6.3 첨판 폭·두께 = 5.3과 동일
- 6.5 첨판 두께 조정·길이 = 5.5와 동일
- 6.6 소요전단강도 = 5.6과 동일

### 6.4 플랜지 볼트 설계지압강도 및 배열 ⭐(핵심 차이)
설계지압강도 = **아래 3종류 공칭강도 중 최솟값 × φ(=0.75)**

| # | 파괴모드 | 공칭강도 Rn | 비고 |
|---|---|---|---|
| (1) | 볼트 자체 전단 | `Rn = Fnv·Ab·Ns` | Fnv=0.50Fu, Ab=πd²/4, Ns=전단면수 |
| (2) | 모재 플랜지 지압 | `Rn = 1.2·Lc·tf·Fu ≤ 2.4·d·tf·Fu` | Lc=순간격, Lc1(연단인접)·Lc2(중간부) 구분 |
| (3) | 첨판 지압 | `Rn = 1.2·Lc·t·Fu ≤ 2.4·d·t·Fu` | t=외+내첨판 두께 합 |

- **요구 행수**: 각 모드별로 산정 (연단인접 볼트 강도 Rn1, 중간부 볼트 강도 Rn2 반영)
  > `n = 1 + (Puf − m·φRn1)/(m·φRn2)` 형태로 산정 → 3종류 중 **가장 큰 n을 올림**하여 설계행수
  - 공칭폭 300mm(엇모)는 0.5단위 올림
- 참고: 전강도설계는 모재 지압강도가, 부분강도설계는 첨판 지압강도가 작아질 수 있음

### 6.7 고력볼트 배열·첨판 춤·두께 가정
웨브 설계지압강도 산정을 위해 배열·첨판 춤·두께를 **먼저 가정**한다.

### 6.8 웨브 볼트 설계지압강도와 배열
- **6.8.1** 설계지압강도 = 3종류 중 최솟값 × 0.75 (플랜지와 동일 방식, **열수 방향**으로 요구개수 산정)
  - ⚠ **모재 웨브는 연단부가 없으므로** Lc1·Rn1(연단인접 항) 해당 없음 → 중간부 항만 적용
- **6.8.2** 3종류 중 가장 큰 요구 열수를 올림하여 설계 열수 m 결정

### 6.9 웨브 첨판 춤·두께 검토와 너비 산정
- 설계된 행수·열수를 **가정값과 비교** → 다르면 재가정·재수행(루프)
- 일치 시 춤·두께 확정, **너비 산정은 마찰접합과 동일**

## 제7장 기둥의 마찰접합부 설계

보(5장)와 골격은 같으나 **소요력이 휨 → 압축**으로 바뀐다. 볼트는 **미끄럼강도**(7장), 부분강도비 **α = 1.0/0.75/0.5**.

### 7.1 설계 순서 개요
**[플랜지 이음]** ① 플랜지 소요압축강도 → ② 첨판 소요단면적 → ③ 첨판 폭·볼트지름·열수 → ④ 첨판 두께 → ⑤ 설계미끄럼강도·요구행수 → ⑥ 볼트 배열·개수 → ⑦ 첨판 두께 조정 → ⑧ 첨판 길이
**[웨브 이음]** ① 웨브 소요압축강도 → ② 볼트지름 → ③ 설계미끄럼강도·요구개수 → ④ 웨브 춤 볼트배열 → ⑤ 첨판 춤·두께 → ⑥ 첨판 너비

### 7.2 플랜지 소요압축강도 ⭐(핵심 차이)
> `Puf = α · φc · bf · tf · Fy`  (α=1.0/0.75/0.5, φc=0.9, bf=플랜지폭, tf=플랜지두께, Fy=형강 항복강도)

### 7.3 플랜지 첨판 폭·두께 — **5.3과 동일**
- 7.3.1 소요단면적 `Aupf = Puf/(φ·Fy)`, 7.3.2 폭=공칭폭, 7.3.3 두께 산정식 동일

### 7.4 플랜지 볼트 설계미끄럼강도·배열 — **5.4와 동일**
- 7.4.1 `φRn = φ·μ·hsc·To·Ns`, 7.4.2 열수 [표 7.1](=표 5.1과 동일), 7.4.3 요구행수 `n = Puf/(m·φRn)` 올림

  **[표 7.1] = [표 5.1]과 동일** (100·125→M16 2열 / 150~250→M20 2열 / 300→M22 4열 엇모 / 350·400→M22 4열)

### 7.5 플랜지 첨판 두께 조정·길이 — **5.5와 동일** (기둥 이음 틈 10mm 포함)

### 7.6 웨브 소요압축강도 ⭐
- 압연 H형강: `Pauw = α · φc · (H·tw + 0.86·r²)·Fy` (웨브+필렛 단면)
- 용접 H형강: `Pauw = α · φc · H·tw · Fy` (필렛 미포함)

### 7.7 웨브 볼트 설계미끄럼강도 — 플랜지와 동일

### 7.8 웨브 볼트 개수·배열 — **5.8과 동일 구조**
- 7.8.1 `mn = Pauw / φRn`
- 7.8.2 배치조건 `(m−1)·Pc + (60 + tf + r)·2 ≤ H`, Pc=60/90/120, 선택규칙 동일

### 7.9 웨브 첨판 춤·두께·너비 — **5.9와 동일**
- 춤 `dpw = (m−1)·Pc + 2·40`, 두께(양면동일) `= 0.5·Pauw/(φc·Fy·dpw)`, 너비 `= 2·[(n−1)·60 + 2·40] + 10`

## 제8장 기둥의 지압접합부 설계

**7장(압축 지배) + 6장(지압강도)** 의 결합. 부분강도비 α = 1.0/0.75/0.5.

### 8.1 설계 순서 개요
**[플랜지 이음]** ① 플랜지 소요압축강도 → ② 첨판 소요단면적 → ③ 첨판 폭·볼트지름·열수 → ④ 첨판 두께 → ⑤ **설계지압강도·요구행수** → ⑥ 볼트 배열·개수 → ⑦ 첨판 두께 조정 → ⑧ 첨판 길이
**[웨브 이음]** ① 웨브 소요압축강도 → ② 볼트지름·**배열 가정** → ③ 웨브첨판 춤·두께 **가정** → ④ 설계지압강도·요구 행수 → ⑤ **배열 검토(다르면 ②부터 재수행)** → ⑥ 첨판 춤·두께 결정 → ⑦ 너비

### 8.2 / 8.3 — **마찰접합(7장)과 동일**
- 8.2 플랜지 소요압축강도 = 7.2 (`Puf = α·φc·bf·tf·Fy`)
- 8.3 첨판 폭·두께 가정 = 7.3

### 8.4 플랜지 볼트 설계지압강도·배열 — **6.4와 동일**
- 3종류(볼트전단 `Fnv·Ab·Ns` / 모재지압 `1.2Lc·tf·Fu ≤ 2.4d·tf·Fu` / 첨판지압 `1.2Lc·t·Fu ≤ 2.4d·t·Fu`) 최솟값 × 0.75
- 요구 행수 3종 중 최대 → 올림 (300mm 엇모는 0.5단위)

### 8.6 웨브 소요압축강도 — **7.6과 동일**(압연 필렛 포함 / 용접 미포함)

### 8.7 배열·첨판 춤·두께 가정 → 8.8 웨브 볼트 설계지압강도·배열
- 6.8과 동일: 3종류 최솟값×0.75, **모재 웨브는 연단부 없음**(Rn1 미해당), 요구 열수 산정

### 8.9 웨브 첨판 춤·두께 검토와 너비
- 가정값과 비교 → 다르면 재수행, 일치 시 확정, 너비는 6/7장과 동일
