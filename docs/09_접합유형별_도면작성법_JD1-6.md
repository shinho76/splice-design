# 접합부 유형별 CAD 도면 작성법 (JointDetailDWG.exe · 12/13/14.dxf 역분석)

> 출처: `JointDetailDWG.exe` 실제 출력 DXF(12=보이음, 13=기둥이음, 14=6유형 통합시트)를 ezdxf로 전수 파싱.
> 목적: exe가 **접합부 6종**을 각각 어떻게 그리는지 규칙화하고, 현재 웹서비스에 이식할 때 필요한 항목 도출.

## 0. 공통 규칙 (6종 전부 동일)

| 항목 | 사양 |
|---|---|
| **좌표 스케일** | 1 도면단위 = **20 mm** (피치 60mm = 3.0u). 부재를 **고정 프레임(100×125u ≈ 2000×2500mm)** 에 맞춰 배치 |
| **레이어(색)** | `MAIN`(7 외형) · `FLG_PL`(3 녹, 플랜지첨판) · `WEB_PL`(4 청, 웨브첨판) · `BOLT`(6 마젠타, 평면볼트) · `VER_BOLT`(1 적, 입면볼트축) · `STIFF`(스티프너) · `DIM`(7 치수·문자) · `MINI_BOX`(7 테두리·정보표) |
| **볼트 심볼** | 평면 = **구멍원(r≈0.54u) + 중심십자**(모두 BOLT). 입면 관통 = **VER_BOLT 축선** |
| **첨판** | **외곽선(LWPOLYLINE)만, 해칭 없음**. 표기 `G.PL. t x w x L x nEA` |
| **치수** | AutoCAD **DIMENSION** + 각 분할점 **witness(연장선)** + **DOT_FILLED(채운 점)** 종단. 문자 **1.0u=20mm** |
| **문자** | 도면 콜아웃·치수 = 20mm(1.0u) · 정보표/제목 = 35mm(1.75u) |
| **부재 연장/파단** | 이음부 밖으로 부재를 연장하고 **양 끝 Z자 파단선**(보=좌우, 기둥=상하) |
| **하단 MINI_BOX** | `Title|JD번호|단면|강종` + 유형별 판·볼트 행 |
| **표기 포맷** | 볼트 `n-M20 H.T.B` · 단면 `H-HxBxt1xt2` · 스티프너 `STIFF. PL. 20t` · 용접 백킹 `Back Bar` |

## 1. 6가지 접합유형 (JD1 ~ JD6)

14.dxf는 6종을 **3열×2행 고정프레임 그리드**로 배치. 편람 4.x 접합형식과 일대일 대응.

### JD1 — 기둥이음 (Column Splice) · 모멘트접합
- **구성**: 웨브 입면(좌) + 플랜지 평면(우), 부재 **세로** 배치
- **판**: 웨브첨판 `G.PL.12x320x285x2EA` + 외플랜지첨판 `12x195x405x2EA` + 내플랜지첨판 `12x70x405x4EA`
- **볼트**: Web 10-M20, Flg 6-M20 · 레이어 MAIN/BOLT/**VER_BOLT**/WEB_PL/FLG_PL 전부 사용
- **특징**: 전강도 모멘트접합(플랜지+웨브 모두 첨판). 기둥이라 상하 파단선

### JD2 — 보이음 (Beam Splice) · 모멘트접합
- **구성**: 웨브 입면(상) + 플랜지 평면(하), 부재 **가로** 배치 *(= 12.dxf)*
- **판**: 웨브첨판 `8x380x165x2EA` + 외/내 플랜지첨판(`12x195x405x2EA` / `12x70x405x4EA`)
- **볼트**: Web 6-M20, Flg 6-M20 · VER_BOLT 사용
- **특징**: JD1과 동일 구조, 방향만 가로

### JD3 — 전단: 큰보+작은보 (Shear, 스티프너)
- **구성**: 작은보(H-500)가 **큰보/기둥(H-900x300)** 에 프레이밍되는 **T자** 형상
- **판**: **웨브 전단탭 1매** `G.PL.20x320x287x1EA` + **스티프너판 `STIFF. PL. 20t (3SIDES)`**
- **볼트**: Web 10-M20 (전단볼트만) · 레이어 MAIN/WEB_PL/BOLT/**STIFF** — **플랜지첨판·VER_BOLT 없음**
- **특징**: 전단접합(모멘트 없음). 지지부재에 **스티프너 3면용접**. `ARC`로 작은보 **코핑(절취)**

### JD4 — 전단: 기둥플랜지+보 (Shear, 단순)
- **구성**: 보가 **기둥 플랜지**에 직접 접합 (지지부재 세로)
- **판**: 웨브 전단탭 1매 `G.PL.20x320x160x1EA` (**짧음**)
- **볼트**: Web 10-M20 · 레이어 MAIN/BOLT — **WEB_PL·STIFF·FLG_PL 없음**(가장 단순)
- **특징**: 스티프너 없이 기둥 플랜지에 탭 용접 후 보 웨브 볼트

### JD5 — 전단: 기둥웨브+보 (Shear, 웨브첨판)
- **구성**: 보가 **기둥 웨브**(또는 큰보)에 웨브첨판으로 접합
- **판**: 웨브 전단탭 1매 `G.PL.20x320x287x1EA`
- **볼트**: Web 10-M20 · 레이어 MAIN/WEB_PL/BOLT — STIFF 없음
- **특징**: JD3에서 스티프너를 뺀 형태

### JD6 — 보이음 (상부용접) (Beam Splice, Top Welded)
- **구성**: JD2와 유사(가로 보이음)이나 **상부 플랜지는 용접**
- **판**: 웨브첨판 `8x380x165x2EA` + 외/내 플랜지첨판 + **`Back Bar`(용접 백킹바) ×2**
- **볼트**: Web 6-M20, Flg 6-M20 (VER_BOLT 18개 = JD2의 절반 → **상부는 볼트 없이 용접**)
- **특징**: 하부 플랜지·웨브는 볼트, **상부 플랜지는 백킹바 대고 완전용입 용접**

## 3. 현재 웹서비스에 그대로 구현하려면 — 추가 필요 항목

현재 앱은 **보이음/기둥이음(모멘트, 마찰·지압)** 2종만 지원. 6종 전체를 만들려면:

### A. 설계 엔진 (engine)
1. **전단접합 설계 3종**(JD3·4·5): 전단력만으로 웨브 전단탭 볼트수·판두께 설계 + **편심모멘트(e)** 고려. 지지형식별(큰보/기둥플랜지/기둥웨브) 차이.
2. **스티프너판 설계**(JD3): 전단탭 지지 스티프너 두께·치수 + **3면 용접** 각장.
3. **상부용접 보이음**(JD6): 상부 플랜지 **완전용입 용접(CJP)** 설계 + 하부 볼트, **백킹바** 규격.
4. **탭 용접 설계**(JD4·5): 전단탭↔지지부재 필릿용접 각장.
5. **작은보 코핑(절취) 산정**(JD3): 큰보 플랜지 회피 절취 치수.

### B. 데이터·타입 (types)
1. `JointType` 확장: `보이음 | 기둥이음 | 전단:큰보+작은보 | 전단:기둥플랜지+보 | 전단:기둥웨브+보 | 보이음(상부용접)` *(마찰/지압과는 별개 축)*
2. **지지부재 입력** `supportSection`(전단 3종은 H-900 등 지지부재 필요)
3. 결과 확장: `stiffener?`, `weld?`(각장·형식), `backBar?`, `webTabEA(1|2)`, `coping?`

### C. UI
1. **접합유형 선택기**(6종) — 부재 토글을 유형 선택으로 대체/확장
2. **지지부재 선택**(전단 유형에서만 노출)
3. **결과표 열 적응**: 전단유형은 플랜지첨판 열 숨김 + 스티프너/용접 열 표시

### D. 도면(DXF/SVG) 생성기
1. **`STIFF` 레이어 + 스티프너 사각형 + 3SIDES 용접기호**
2. **전단 T자 배치**(지지부재 + 보 프레이밍) — 현재 이음(가로·세로)과 별도 레이아웃
3. **용접기호**(상부용접 CJP, 필릿) + **`Back Bar`** 표기
4. **코핑 ARC**(작은보 절취 곡선) — 현 생성기엔 ARC 없음
5. **웨브첨판 1EA/2EA 구분** (전단=1, 모멘트=2)
6. **기둥 세로배치**(JD1) + **고정프레임 fit-to-frame** + **다중상세 그리드 시트**(JD1~6 한 장)
7. MINI_BOX 정보표 **유형별 행 구성**(모멘트=7행 / 전단=3행)

### E. 우선순위 제안
1. **(1단계)** 타입/유형 선택기 + 전단접합(JD5 웨브첨판) 설계·도면 — 가장 수요 큰 단순전단
2. **(2단계)** JD3 스티프너·코핑, JD4 기둥플랜지, 지지부재 입력
3. **(3단계)** JD6 상부용접·백킹바·용접기호, 기둥 세로배치·고정프레임·그리드 시트
