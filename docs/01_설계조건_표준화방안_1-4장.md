# 고력볼트 표준접합 설계편람 — 설계조건 · 표준화 방안 (1~4장)

> **원전**: 한국강구조학회 『KBC-09 강구조기준에 따른 고력볼트 표준접합 설계편람』 (KSSC-M-09-03, 2009)
> **범위**: 제1장 재료 · 제2장 표준접합부의 일반사항 · 제3장 첨판/볼트배치 표준화 · 제4장 표준접합 원칙
> **성격**: 5~8장 "설계 프로시저"에 들어가기 전에 반드시 알아야 할 **재료·강도 조건**과 **표준화 규칙**을 정리한 문서 (= "왜 이렇게 설계하는가")

## 0. 편람의 배경과 기본 사상

| 항목 | 내용 |
|---|---|
| 설계법 | **KBC-09 한계상태설계법(LSFD)** 적용 (구 KBC-05 허용응력 아님) |
| 새 볼트 | 기존 F10T 대비 성능 1.3배인 **F13T(1,300MPa급)** 도입 → 볼트 소요개수 감소 → 자재비·가공비·시공비 절감 |
| 접합방식 | **마찰접합 + 지압접합** 모두 규정 (KBC-05는 마찰접합만) |
| 구멍 | 본 편람은 **표준구멍**만 다룸 (과대·슬롯구멍은 예제에서 해당 부분만 변경) |
| 표준화 목적 | 설계·제작·시공의 **단순성·편리성·품질관리 용이성** |
| 참조 | 일본 「철골구조 표준접합부(H형강편)」를 참조해 다수 단순화 |
| ⚠ 중요 | **KBC-09는 반드시 지켜야 할 규정이지만, 이 편람의 표준화는 강제 규정이 아님** (권장 표준) |

**표준화의 핵심 아이디어 4가지**
1. 플랜지 첨판 폭 = 형강의 **공칭폭**으로 고정
2. 볼트 지름은 **플랜지 폭에 따라** 표준화 (M16/M20/M22만 사용)
3. **구멍간격(피치)·종연단거리를 볼트지름과 무관하게 일정**하게 고정
4. 그 외 여러 항목을 표준화하여 접합부 설계/제작/시공을 단순화

## 제2장 표준접합부의 일반사항

이 장은 **모든 접합 계산의 입력값(볼트 강도·장력·구멍·연단거리)** 을 확정한다.

### 2.1 고력볼트 지름과 표준구멍 크기
- 표준접합에는 **M16, M20, M22만** 사용, **표준구멍만** 적용
- **플랜지 볼트와 웨브 볼트의 지름은 동일**하게 사용
- 구멍중심 간 **응력방향 거리 = 볼트지름과 무관하게 모두 60mm**

**[표 1.5] 볼트 구멍 지름과 응력방향 중심거리 (mm)**

| 호칭 | 볼트 구멍 지름 | 응력방향 중심거리 |
|---|---|---|
| M16 | 18 | 60 |
| M20 | 22 | 60 |
| M22 | 24 | 60 |

### 2.2 고력볼트 접합의 마감처리
- 모든 접합면 마감처리는 **미끄럼계수 μ ≥ 0.5**
- KBC-09에 따라 **블라스팅 후 페인트하지 않은 면은 μ = 0.5** 확보로 간주

### 2.3 설계볼트장력과 표준볼트장력
- 설계볼트장력 `To` = **0.7·Fu × 0.75·Ab** (= 볼트 설계기준인장강도의 0.7배 × 공칭단면적의 0.75배)
- 표준볼트장력(체결작업 적용) = **설계볼트장력 × 1.1**

**[표 1.6] 설계볼트장력 · 표준볼트장력**

| 등급 | 호칭 | 공칭단면적 Ab (mm²) | 설계볼트장력 To (kN) | 표준볼트장력 (kN) |
|---|---|---|---|---|
| **F10T** | M16 | 201 | 106 | 117 |
| | M20 | 314 | 165 | 182 |
| | M22 | 380 | 200 | 220 |
| **F13T** | M16 | 201 | 137 | 151 |
| | M20 | 314 | 214 | 235 |
| | M22 | 380 | 259 | 285 |

### 2.4 마찰접합용 1면전단 미끄럼강도
- 공칭미끄럼강도 `Rn` = KBC-09에 따라 산정 → [표 1.7]
- **설계미끄럼강도 `φRn` = 공칭미끄럼강도 × 0.85** (φ = 0.85)
- 산정식: `φRn = φ · μ · hsc · To · Ns`
  - φ = 0.85, μ = 0.50(블라스트 후 페인트 안 함), hsc = 1.00(표준구멍), To = 설계볼트장력, Ns = 전단면 수

**[표 1.7] 고력볼트 1면전단 공칭미끄럼강도 (kN)**

| 호칭 | F10T (Fu=1,000) | F13T (Fu=1,300) |
|---|---|---|
| M16 | 52.8 | 68.6 |
| M20 | 82.4 | 107.0 |
| M22 | 99.7 | 130.0 |

### 2.5 지압접합용 고력볼트 축부의 전단강도
- ⚠ 표준접합은 **나사부가 전단면에 포함되지 않는 경우로 제한** (포함 시 별도 설계)
- 공칭전단강도 = **설계기준인장강도 Fu의 50%** (`Fnv = 0.50·Fu`)
- **설계전단강도 = 공칭전단강도 × 0.75** (φ = 0.75)

**[표 1.8] 지압접합 공칭전단강도 (MPa)**

| 강도 | F10T | F13T |
|---|---|---|
| 공칭전단강도 Fnv | 500 | 650 |

### 2.6 지압접합용 볼트구멍의 지압강도
- 지압강도는 **첨판·모재 모두** 검토, 사용하중상태에서 **구멍변형이 고려되는 경우**로 함
- 두 가지 파괴모드 검토(둘 중 작은 값):

| 파괴모드 | 공칭지압강도 | φ |
|---|---|---|
| ① 판재 전단파괴 | `Rn = 1.2 · Lc · t · Fu` | 0.75 |
| ② 판재 국부압축파괴 | `Rn = 2.4 · d · t · Fu` | 0.75 |

  - `Lc` = 응력방향 순간격, `t` = 판두께, `d` = 볼트 공칭지름
- 다음일수록 **볼트 축부 전단강도보다 구멍 지압강도가 지배**(작아짐): ① F13T(고강도볼트) ② M22(큰지름) ③ SS400(약한 피접합재) ④ 얇은 피접합재

### 2.7 종연단거리
- 연단부 파괴 방지를 위해 최소거리 확보. KBC-09는 가공방법별로 규정하나(표 1.9), **표준접합은 응력방향 종연단거리를 모두 40mm로 통일**

**[표 1.9] 볼트중심~연단 최소거리 vs 표준접합 종연단거리 (mm)**

| 호칭 | 전단절단·수동가스절단 | 압연형강·자동가스절단·기계마감 | 표준접합 종연단거리 |
|---|---|---|---|
| M16 | 28 | 22 | **40** |
| M20 | 34 | 26 | **40** |
| M22 | 38 | 28 | **40** |

### 2.8 사용성 마찰접합
- 강도한계상태는 지압 또는 마찰 **선택** 가능하나, **사용성한계상태는 반드시 마찰접합을 만족**해야 함
- 판정: **사용하중/계수하중 비율 ≥ 1.43** → 사용성 마찰접합 검토 불필요(강도한계상태가 지배)
- 근거(해설):
  - 강도한계 지압 설계전단강도 `φRn = 0.75 × 0.5·Fu·Ab = 0.375·Fu·Ab`
  - 사용성 마찰 설계강도 `= 1.0 × 0.5 × 0.7·Fu × 0.75·Ab = 0.263·Fu·Ab`
  - 비율 = 0.375 / 0.263 ≈ **1.43**
- 비율이 1.43 미만이어도 F13T·M22·SS400·얇은 판일수록 지압강도가 접합부 설계를 지배하는 경향

## 제3장 첨판 표준화와 고력볼트 배치의 표준화

### 3.1 첨판 표준화
- 혼선을 줄이기 위해 첨판 폭·두께 사용범위를 일정하게 규정
- **플랜지 첨판 폭 = 형강의 공칭폭**
- 첨판 두께는 규격화(아래 표)
- 내·외첨판이 쓰이는 플랜지 이음: 단면적을 동일하게 두는 방식이 아니라 **(요구 볼트강도 − 외측 확보 볼트강도 = 내측 요구 볼트강도)** 개념으로 외첨판 단면적 산정

**[표 3.1] 플랜지 첨판 두께 표준 (mm)**: 9, 12, 16, 19, 22, 25, 28
**[표 3.2] 웨브 첨판 두께 표준 (mm)**: 6, 9, 12, 14, 16, 19

### 3.2 고력볼트 배치 표준화

#### 3.2.1 플랜지 이음
- 응력방향 **피치: 정렬배치 60mm / 엇모배치 45mm** (M22 이하 볼트, 지름 무관)
- 응력방향 **연단거리 = 40mm** (지름 무관)
- 형강 이음부 **이격거리 = 10mm** (단, 기둥 밀착접합 metal touch은 0mm)
- 플랜지 공칭폭별 첨판폭·게이지·볼트규격·열수·배치 → [표 3.3]

**[표 3.3] 플랜지 이음부 규격 (요약)**

| 플랜지 공칭폭 | 볼트 규격 | 볼트 열수 | 배치 |
|---|---|---|---|
| 100, 125 | M16 | 2열 | 정렬 |
| 150, 175, 200, 250 | M20 | 2열 | 정렬 |
| 300 | M22 | 4열 | **엇모** |
| 350, 400 | M22 | 4열 | 정렬 |

> 볼트와 첨판의 최소거리는 60mm ([그림 3.3]).

#### 3.2.2 웨브 이음
- 웨브 볼트 배열은 **상하 대칭**, 웨브 첨판 상하길이는 **부재 춤의 60% 이상** 원칙
- 웨브 제1열 볼트~내첨판 간격 **60mm 이상**(미만이면 웨브/플랜지 볼트를 절반 피치로 엇갈림, [그림 3.4])
- 웨브 볼트 **상하방향 피치: 60 / 90 / 120mm** 3종, **종방향 피치: 60mm**
- 종·횡 연단거리 = **40mm** (지름 무관)
- 이격거리: 보 이음 10mm, 기둥 이음 0mm
- 웨브 볼트 배치는 6가지 양상 → [표 3.4]

| (c) | 1열 | 90 | 60 |

## 제4장 표준접합 원칙

접합부 **소요강도의 기준(전강도/부분강도)** 과 **공칭강도 산정 원칙**을 정한다.

### 4.1 보의 표준접합 원칙

#### 4.1.1 전강도와 부분강도
- 휨강도에 대해 **전강도설계(Full Strength, 100%)** 또는 **부분강도설계(Partial Strength, 85% / 70%)** 로 표준화
- 부분강도 선택 근거(구조설계자 판단):
  1. 보 단부 전강도 설계휨강도 대비 실제 소요강도 비율
  2. 기둥에서 내민 브라켓(bracket) 길이 고려 → 기둥면 모멘트 대비 이음부 감소 모멘트 비율
  3. 1)·2) 동시 고려
- 웨브(전단) 이음도 동일하게 **100% / 85% / 70%** 로 표준화

> **보의 부분강도비 α = 1.0 / 0.85 / 0.7**

#### 4.1.2 휨재의 공칭휨강도
- (1) **횡좌굴에 의한 강도 감소 무시**, **국부좌굴만 고려**한 공칭휨강도로 설계
- (2) **순단면 무시, 총단면 공칭휨강도**로 설계
- 해설(인장 플랜지 파단 검토, KBC-09 0706.13):
  - `Fu·Afn ≥ Yt·Fy·Afg` → 순단면 파단 무시, **총단면 공칭휨강도** 사용
  - `Fu·Afn < Yt·Fy·Afg` → 순단면 파단 공칭휨강도 검토 필요
  - `Yt = 1.0 (Fy/Fu < 0.8)`, `Yt = 1.1 (그 외)` — `Afg`: 인장플랜지 총단면적, `Afn`: 순단면적
- **국내 강재는 [표 4.1]과 같이 Fy/Fu가 모두 0.8 미만 → Yt = 1.0** 사용

**[표 4.1] 강재별 Fy/Fu 비율과 Yt**

| 강재 | SS400 | SM490 | SM520 |
|---|---|---|---|
| Fy | 235 | 325 | 355 |
| Fu | 400 | 490 | 520 |
| Fy/Fu | 0.588 | 0.663 | 0.683 |
| Yt | 1.0 | 1.0 | 1.0 |

> 결론: [표 4.2] 검토 결과 **대부분의 경우 순단면 파단 검토 없이 총단면 공칭휨강도 사용 가능** (SM490 일부 폭에서만 근소하게 부족하나 실무상 무시 가능).

### 4.2 기둥의 표준접합 원칙

#### 4.2.1 전강도와 부분강도
- **전강도설계** 또는 (지압접합·밀착접합 고려) **전강도의 75% / 50% 부분강도설계**로 표준화
- 기둥이 조합력(압축+휨)을 받아도 별도 고려 없이 **공칭압축강도 기준으로 표준화**
- 부분강도 선택 근거: ① 전강도 설계압축강도 대비 실제 소요강도(양축 휨 동시 고려) 비율 ② 밀착접합(metal touch) 효과 ③ 동시 고려

> **기둥의 부분강도비 α = 1.0 / 0.75 / 0.5**

#### 4.2.2 기둥의 공칭압축강도
- (1) **세장비 무시**(= 항복만 고려)한 공칭압축강도로 설계
- (2) **총단면적** 적용
- (3) 압연형강 총단면적 = 압연 **필렛 포함** / 용접형강 = 용접 **필렛 미포함**
- (4) 플랜지 첨판은 **플랜지 단면적**을, 웨브 첨판은 **웨브+필렛 단면적**을 부담

### ⚠ 설계 시 주의점
1. **표준화는 권장 규칙**이며 KBC-09 규정 자체가 아니다. 과대·슬롯구멍, 나사부 전단면 포함 등은 표준접합 적용 불가 → 별도 설계.
2. 볼트 지름은 **플랜지 폭이 결정**한다(표 3.3). 웨브 볼트도 같은 지름을 쓴다.
3. 피치·연단거리·구멍중심거리는 **볼트지름과 무관하게 고정값**이므로 도면·제작이 단순해진다.
4. 사용성 마찰 검토(2.8)는 지압접합을 선택했어도 **하중비 < 1.43이면 반드시 별도 확인**.
5. 다음 단계(5~8장)로 넘어가기 전, 부재가 **보(휨 지배)** 인지 **기둥(압축 지배)** 인지, **마찰**인지 **지압**인지를 먼저 확정한다.
