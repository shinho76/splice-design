## 3. 계산 엔진 (src/engine/)

4개 접합유형 전부 구현. **핵심 소요력 공식이 모두 부록과 일치**.

| 유형 | 소요력 공식 | 볼트 설계강도 |
|---|---|---|
| 보 마찰(5장) | Mu = α·φ·Mn (국부좌굴), Puf = Mu/dm | 설계미끄럼강도 φRn (φ0.85) |
| 보 지압(6장) | 동일 | 6.4.1 **연단 Rn₁/중간 Rn₂**(전단파괴 1.2Lc·t + 국부압축 2.4d·t) |
| 기둥 마찰(7장) | **Nc = α·φc·bf·tf·Fy**, Vw = α·φc·H·tw·Fy | 미끄럼강도 |
| 기둥 지압(8장) | 동일 | 지압강도 + 정렬세칙 |

**주요 발견**
- **단면성능 주입 무효**: 기둥 Nc·Vw는 순수 기하(bf·tf, H·tw)라 KS Zx/Sx와 무관 → "9.0 두께 경계"는 KS 주입으로 해소 안 됨(안전측 ±1등급).
- **KS D 3502 원표 주입 완료**: 73종 중 70종 확정 단면성능(Ag·Sx·Zx·r), 단위변환(cm²×100, cm³×1000). 검증: H-386x299 Zx=1,920,000 = 9장 예제 일치.
- 지압 웨브(6.8): 중간부 지압강도 φRn₂에 **전단파괴 1.2·(60−hole)·tw** 항 추가로 얇은 web 재현.

## 4. 검증 결과 (부록 설계표 전수 대조)

| 조건 | 전항목 일치 |
|---|---|
| 보 마찰 100% SHN490 F10T | **69/73** |
| 보 지압 100% SHN490 F10T | **63/73** |
| 기둥 마찰 100% SHN490 F10T | 50/73 (축력 72/73) |
| 기둥 지압 100% SHN490 F10T | 34/73 (축력 72/73) |
| 보 마찰 **85%**(α 검증) | 17/19 |
| 보 마찰 **F13T**(볼트 검증) | 18/19 |

- 잔여 불일치는 **첨판 두께 반올림 경계·두꺼운 web ±1등급·광폭박판 이상치**로, 로직 오류 아님(안전측).
- α·볼트등급 파라메트릭 **일반화 입증**(플랜지 19/19).
- 재현: `node src/engine/validate.ts` / `validate_col.ts` / `validate_variants.ts`

## 5. 웹서비스 기능 (src/components/)

| 기능 | 설명 |
|---|---|
| 목록표 | 부록 설계표 재현(필터: 부재·접합·강종·볼트·α). α는 50~100% 드롭다운+직접입력. 1000+ 콤마·시리즈/항목 테두리 |
| **전문 계산서** | 표제부 + **목차** + **1./1.1 번호 체계** + 결과요약표 (제9장 예제 형식) |
| **접합 상세도(SVG)** | 웨브 입면(외·**내첨판** 단면)+플랜지 평면(웨브 은선). 치수를 **실좌표에서 도출**→항상 일치, 엇모 연단 40mm 확보, 해칭·다크모드 |
| **DXF 출력** | AutoCAD R12(해칭·치수·은선·레이어 8종), ezdxf 렌더 검증 |
| **PDF 출력** | 인쇄 전용 CSS로 계산서만 깔끔히(window.print) |
| **물량산정** | 부재별 볼트·첨판중량(7,850kg/m³) 집계, CSV·**Excel(.xlsx, SheetJS)** |
| **프로젝트** | 다중부재(단면+조건) 담기·합산물량·localStorage·JSON 저장/불러오기 |
| UI/UX | 모던 게임풍, 다크/라이트, 반응형 |
