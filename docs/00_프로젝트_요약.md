## 2. UI — 관제 콘솔 대시보드 (시안 ① 적용)

디자인 시안 4종(관제콘솔·벤토·리포트·워크벤치) 중 **①관제 콘솔** 채택·적용.

- **레이아웃**: 좌측 아이콘 레일(검토결과·물량·프로젝트·전체DXF) + 상단바(브랜드·필터·조건요약·테마토글) + **KPI 스트립 5칸**(검토부재·적합·부적합·고력볼트·강재물량) + 본문 2분할(결과표 + **우측 실시간 상세패널**) + 하단 상태바
- **상세패널**: 부재 선택 시 스펙(휨/압축·플랜지/웨브 볼트·외/내첨판) + 접합상세 SVG + `상세계산서 / DXF / 프로젝트 담기` 버튼
- **테마**: **다크 기본**(청사진 네이비) / 화이트(규격서 팔레트). 상단 `☾다크·☀화이트` 세그먼트 토글, `localStorage` 선택 기억. 토큰 레벨(`:root`·`@media`·`[data-theme]`)로 두 테마 완전 대응
- **결과표**:
  - `table-layout: fixed` + `<colgroup>` → **보/기둥 전환 시 열 테두리 위치 불변**
  - 3열 구성: **단면치수 · r · 단위중량**(카탈로그값/단면적환산)
  - **호칭치수(PDF)** 로 시리즈 그룹핑(굵기 아닌 색으로 구분)
  - sticky 헤더: `border-collapse: separate` + 불투명 셀 + 헤더 오프셋 정렬 → **스크롤 비침·테두리 흔들림 없음**, **전 테두리 1px 통일**(그리드선 없음)
  - 필터 α: 50·60·65·70·75·80·85·90·95·100%

## 3. 계산 엔진 (src/engine/)

4개 접합유형(보/기둥 × 마찰/지압) 구현, **핵심 소요력 공식이 부록과 일치**.

| 유형 | 소요력 | 볼트 설계강도 |
|---|---|---|
| 보 마찰(5장) | Mu=α·φ·Mn, Puf=Mu/dm | 설계미끄럼강도 φRn |
| 보 지압(6장) | 동일 | 6.4.1 연단 Rn₁/중간 Rn₂ |
| 기둥 마찰(7장) | **Nc=α·φc·bf·tf·Fy**, Vw=α·φc·H·tw·Fy | 미끄럼강도 |
| 기둥 지압(8장) | 동일 | 지압강도 |

- KS D 3502 원표 70/73 주입, 3 보강단면은 편람 Mu 캘리브레이션
- 검증(부록 전수): 보마찰 69/73·보지압 63/73·기둥축력 72/73, α·F13T 파라메트릭 일반화 입증
- `hbeam_catalog.ts`: 호칭치수·단위무게(현대제철 Products Guide, KS D 3502 Metric)

## 4. 도면(DXF/SVG) — JointDetailDWG.exe 표현 규약 재현

레거시 `JointDetailDWG.exe`(2001, MFC) 출력 DXF(**12=보이음·13=기둥이음·14=6유형 통합**)를 ezdxf로 역분석 → `dxf.ts`가 규약을 그대로 재현. **엔진 무관, 도면 표현만**.

**공통 규약** (exe 샘플 ezdxf 실측 반영, docs/10)
- 버전 **AC1009(R12)** · 좌표 실척(mm) · 레이어 `MAIN(7)·FLG_PL(녹3)·WEB_PL(청4)·BOLT(마젠타6)·VER_BOLT(적1)·DIM(7)·MINI_BOX` — **선두께는 색상펜(CTB)**
- 볼트 = **구멍원+중심십자**(BOLT), 입면 = **볼트축선**(VER_BOLT), 첨판 = **외곽선(해칭 없음)**
- **정식 DIMENSION 엔티티**(+지오메트리 블록) · **이중치수**(세부 체인+전체) · **지시선(leader)** · **화살촉 `_ARCHTICK` 45° 틱(scale 5)** · 폰트 **`OpenSansCondensed-Light.ttf`**(exe 실측)
- **부재 연장 + Z자 파단선**, **외곽 테두리**, 하단 **MINI_BOX 정보표**(전 항목 영문)
- 표기 `n-M20 H.T.B` · `G.PL. txwxLxnEA` · `H-HxBxt1xt2` · `STIFF. PL. 20t`
- **고품질 승격(선택)**: `tools/dxf_gold.py` → R2018 + 객체 선두께(ezdxf)
- **3D 뷰어(Three.js)**: `connParts.ts`(공유 지오메트리)+`ThreeViewer.tsx` — 단면치수 옆 `3D`버튼/상세패널에서 H형강(필렛R 압출)·첨판·볼트(머리·너트·여장) 실시간 렌더
- **IFC 내보내기**: `ifcOut.ts` → IFC4 STEP(H형강=IfcIShapeProfileDef 필렛R, 첨판=사각압출, 볼트=원형압출). web-ifc로 검증(`tools/verify_ifc.mjs`)

**배치**
- **보이음 = 가로배치**(웨브입면 상 + 플랜지평면 하)
- **기둥이음 = 세로배치**(90° 회전변환; 웨브입면 좌 + 플랜지평면 우, 상하 파단)

**정직한 한계**: 볼트 개수·판치수는 **KBC 엔진값**(exe는 AIK-ASD83) → 도면 스타일은 일치하나 수치는 다를 수 있음.

## 6. 기능 (src/components/, engine/)

| 기능 | 설명 |
|---|---|
| 결과표 | 부록 재현(필터: 부재·접합·강종·볼트·α) |
| 전문 계산서 | 표제부+목차+1./1.1 번호+요약표 (제9장 형식) |
| 접합상세도 | SVG(웨브입면·플랜지평면·요약표) + **DXF(exe 규약·정석 DIMENSION)** |
| 물량산정 | 볼트 본수+**중량(KS B 1010 세트)**·첨판중량(7,850kg/m³) 집계, CSV·Excel(SheetJS) |
| 볼트사양 | **표준길이**(그립+부가길이→5mm올림)·**세트중량**(볼트+너트+와셔2매) 플랜지/웨브 개별. `bolt_spec.ts`= **KS B 1010/JIS B 1186 세트중량표**(길이별 gr) 직접 조회. 개별부재 상세패널 + KPI(고력볼트 본수·총중량) 표기 |
| 프로젝트 | 다중부재 저장/불러오기(localStorage·JSON) |
| 출력 | 개별/전체 **DXF**, **IFC4**(web-ifc 검증), **3D(Three.js)** 뷰어, PDF(인쇄) |
