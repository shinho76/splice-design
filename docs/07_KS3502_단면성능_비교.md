# KS D 3502 단면성능 vs 계산값 비교

> **목적**: 웹서비스 엔진이 사용하는 단면성능(치수 기반 계산값)이 KS D 3502 규격값과 얼마나 차이나는지 단면별로 정리.
> **대상**: 부록 「보 100% SHN490 F10T」 압연 H형강 73종.

---

## 1. 방법과 전제

- 엔진은 단면성능을 **치수(H·B·tw·tf·r)에서 기하학적으로 계산**한다(`sections.ts › computeProps`).
- 필렛반경 `r`은 KS 규격표를 직접 보유하지 않으므로, **편람 설계강도표(Mu)에 이분탐색으로 캘리브레이션**하여 결정한다. 편람 Mu는 KS 단면성능으로 산정된 값이므로, **캘리브레이션된 Sx·Zx는 KS 규격값에 준하는 근사(KS-proxy)** 가 된다.
- 아래 표의 **`Δ Sx`** = (캘리브레이션 Sx / 원계산 Sx − 1). 즉 **"KS에 맞춘 값"과 "단순 계산값(r=1.2·tf)"의 차이** = 규격값을 적용하지 않고 순수 계산만 했을 때의 오차 크기.

> ⚠ **주의**: 아래 A·Ix·ix·Sx·Zx는 모두 계산값이다. 프로덕션 정밀도를 위해서는 **KS D 3502 규격표의 확정값을 직접 주입**(propSource='ks')하는 것이 최선이며, 그때의 실제 Δ는 이 표의 Δ Sx와 유사한 크기(대부분 ±1~2%, 광폭 박판·후판에서 최대 ±3~5%)로 예상된다.

**기호**: A=총단면적, 단위무게 w=A×7.85×10⁻³, Ix=단면2차모멘트, ix=단면2차반지름(√(Ix/A)), Sx=탄성단면계수, Zx=소성단면계수.

## 2. 차이 요약

| 구분 | Δ Sx 범위 | 비고 |
|---|---|---|
| 대부분(정상 압연 단면) | **±0.0 ~ ±1.5%** | 계산값으로도 실무상 충분 |
| 얇은 웨브·좁은 플랜지(H-198x99 등) | +1.8 ~ **+3.4%** | 필렛 근사 민감 |
| 광폭 박판(H-388x402x12x12) | **+5.5%** | 필렛/기하 근사 최대 오차 → Mu 재현 불가 원인 |
| 후판 대형(H-428x407x20x35 등) | **−2 ~ −3%** | 후판 필렛 과대계산 경향 |

**결론**: 접합부 설계 결과(볼트 개수·첨판 두께)는 표준값으로 **올림·이산화**되므로 ±1~2% 단면성능 오차는 대부분 흡수된다. 다만 **두께 반올림 경계**에 걸리거나 **광폭 박판**인 경우 결과가 1등급 달라질 수 있어, 정밀 설계 시 KS 규격값 주입을 권장한다.

## 4. 권장 후속 조치
1. **KS D 3502 규격표 CSV/JSON 확보** → `sections.ts`에 `KS_PROPS: {A, w, Ix, ix, Sx, Zx}` 주입, `propSource='ks'`.
2. 주입 후 이 표를 재생성하여 **실제 Δ(규격 vs 계산)** 확정.
3. Δ가 큰 광폭 박판(H-388x402 등)은 우선 규격값 적용으로 Mu·첨판 정밀화.
